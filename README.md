# Real-Time Multi-Room Chat App
Developed by Himanshu Das Biswas (KIIT-DU)

A full-stack messaging platform that allows users to join specific rooms and chat in real-time.

## 🛠 Tech Stack
* [cite_start]**Frontend:** React.js [cite: 12, 21]
* [cite_start]**Backend:** Node.js, Express.js, Socket.io [cite: 13]
* [cite_start]**Database:** MongoDB Atlas [cite: 13, 22]

## 🚀 Key Features
* **Real-time Communication:** Instant message delivery using WebSockets.
* **Typing Indicators:** See when others are typing.
* **Room Based:** Join different rooms for different topics.
* [cite_start]**Secure Auth:** JWT and Bcrypt for safe login[cite: 13, 20].
